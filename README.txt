В файле Config.py заполнить:

client_id = код после создания приложения в яндекс кошельке
redirect_id = сайт на который редиректнет после создания приложения (там будет временный код для получения токена)
bot_token = токен бота телеграм
to = кошелек яндекс на который будут приходить деньги с кошельков в боте
